## Getting Started

### Clone the repo

```bash
git clone https://github.com/your-username/mern-ecommerce.git
cd mern-ecommerce
```

### Install dependencies

```bash
# For frontend
cd frontend
npm install

# For admin panel
cd admin
npm install

# For backend
cd backend
npm install
```

### Run the project

```bash
# Start frontend
cd frontend
npm start

# Start admin panel
cd admin
npm run dev

# Start backend
cd backend
node index.js
```
